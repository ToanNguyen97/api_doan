công việc phiếu thu:
1- load danh sách phòng theo khu phòng => done!
2- check phòng đó có được thuê hay không thì mới hiển thị
3- kiểm tra phòng đó tháng này đã lập phiếu thu chưa => done!
4- nếu chưa lập thì hiển thị nút ghi điện nước, nếu lập thì tắt nút đó
5- khi ghi điện nước xong sẽ lập phiếu thu => done!
6- khi lập phiếu thu sẽ lập luôn cả chi tiết phiếu thu => done!
7- khi lập xong tắt popup sẽ load lại tình trạng là chưa đóng => done!
8- nhấn nút xem chi tiết sẽ chuyển sang trang chi tiết phiếu thu
